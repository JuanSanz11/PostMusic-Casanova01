## Cómo contribuir
1. Haz un fork del proyecto
2. Crea una rama: `git checkout -b mi-feature`
3. Haz commit de tus cambios: `git commit -m 'feat: mi feature'`
4. Haz push a tu fork: `git push origin mi-feature`
5. Abre un Pull Request
